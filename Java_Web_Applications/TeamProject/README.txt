This project was created during the CodingBootcamp6 by Nikos Ksygkis, Alexandros Makris, Belechris Marios and Dimitrios Koulouktis.
It is a web application that allows users to create articles, and interact through comments, like-dislike buttons etc.

There are two roles.

The admin, that publishes an article, edits or deletes users

The simple user that creates articles, comments, likes-dislikes articles, edits personal info.

There is also the option to log in as username: god, password: god where that user
has the option to use the application as an admin or as a user.
